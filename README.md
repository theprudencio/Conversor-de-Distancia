Um programa desenvolvido em C# orientado a objetos via windows form. É um programa básico que permite o usuário uma entrada de valor, que pode ser convertido em KM
ou milhas por escolha do usuário. Nota-se que ainda não há regras de negócio, então não há tratamento de dados.

Uma classe Converte foi criada, onde possui a propriedade e os métodos que calculam a conversão.
